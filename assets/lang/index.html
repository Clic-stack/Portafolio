Habilidades

A medida que el mundo se vuelve más competitivo, desarrollamos habilidades que nos preparan para aprovechar mejores oportunidades.

HTML

El Lenguaje de Marcado de Hipertexto (HTML) es un lenguaje utilizado para crear y estructurar páginas web. Es el lenguaje por excelencia para documentos diseñados con la intención de ser mostrados en la web, gracias a su semántica, que permite definir elementos mediante etiquetas. Estas etiquetas indican cómo debe presentarse el contenido, y HTML suele complementarse muy bien con CSS y JavaScript para lograr una experiencia visual y funcional más completa.

  • Habilidades Desarrolladas:
  • Estructuración de contenido en una página web empleando etiquetas que definen elementos como texto, imágenes, videos, enlaces, tablas, listas, contenido embebido y formularios.
  • Conocimiento e importancia del uso de metadatos en el desarrollo de un sitio web para el posicionamiento SEO (Search Engine Optimization).

CSS3

Las hojas de estilo en cascada (CSS) son un lenguaje que permite diseñar y crear páginas web atractivas para el usuario. Se utiliza para describir la presentación de documentos escritos en HTML o XML. Por ende, cabe recalcar que CSS es una de las tecnologías fundamentales de la World Wide Web, junto con HTML y JavaScript.

  • Habilidades Desarrolladas:
  • Uso y aplicación de las distintas propiedades de CSS para estilar colores, tipografía, enlaces, listas, contenido desbordado, imágenes, posicionamiento y apilamiento.
  • Conocimientos en modelo de caja, visibilidad, unidades de medida, Grid, Flexbox, variables CSS y media queries.

Javascript Backend

JavaScript es un lenguaje de programación multiplataforma orientado a objetos que se utiliza para hacer que las páginas web sean interactivas. Es un lenguaje basado en prototipos, con tipado dinámico, y soporta estilos de programación basados en eventos, funcionales e imperativos. Es ampliamente utilizado en el desarrollo web, aplicaciones del lado del servidor y desarrollo de aplicaciones móviles.

  • Habilidades Desarrolladas:
  • Conocimientos y aplicación de la terminal, variables, tipos de datos (primitivos y estructurados), funciones, métodos en tipos primitivos (números y cadenas), métodos en arreglos, métodos en objetos, operadores básicos y de comparación, coerción de tipos, bucles, condicionales y operadores lógicos.

Javascript Frontend

JavaScript es un lenguaje de programación multiplataforma orientado a objetos, utilizado para dar vida, interactividad y dinamismo a los sitios y páginas web.

  • Habilidades Desarrolladas:
  • Conocimiento y aplicación de transiciones, animaciones, funciones avanzadas (constructoras, de orden superior, de fábrica, recursivas), manipulación del DOM y callbacks.

React

React permite construir interfaces modulares y escalables mediante componentes reutilizables, gestión eficiente del estado y renderizado optimizado con DOM virtual. Su enfoque declarativo y su ecosistema robusto lo convierten en una herramienta clave para el desarrollo de aplicaciones web modernas, especialmente en entornos SPA.

  • Habilidades Desarrolladas:
  • Aplicación de buenas prácticas en React mediante el uso de JSX, estilos modulares y externos, componentes reutilizables, props desestructuradas y gestión eficiente del estado con . Implementación de lógica condicional, estados computados e interacción fluida con el DOM virtual. Dominio de principios de inmutabilidad, operadores modernos como spread y encadenamiento opcional, y uso de métodos funcionales como y para renderizado dinámico.

React

  • Habilidades Desarrolladas:
  • Conocimiento y aplicación de formularios con entradas controladas y no controladas; el hook useEffect; obtención de datos (data fetching); escuchadores de eventos (event listeners); módulos ES (ES Modules); llamadas a API; manejo del estado de carga y gestión de errores; hooks como useMemo y useCallback; metodología BEM; y estilos con Sass.
  • Conocimiento y aplicación del hook useRef, creación de hooks personalizados, componentes funcionales, uso de la prop children, implementación de modales y gestión de errores.
  • Conocimiento y aplicación del hook useReducer, Context API con Provider y Consumer, el hook useContext, React Router, protección de rutas mediante autenticación, gestión de estado global con Zustand y uso de Local Storage.
